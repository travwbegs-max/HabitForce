# HabitForce

A modern, installable Habit Tracker Progressive Web App (PWA) built with **HTML, CSS, and JavaScript**.

No frameworks.

No backend.

No databases.

Everything runs directly from GitHub Pages and stores data locally on the device.

# Installation

Clone the repository.

```
git clone https://github.com/YOUR_USERNAME/HabitTracker.git
```

Open the project.

Enable GitHub Pages.

Visit the website.

On iPhone:

Share

↓

Add to Home Screen

The app is now installed.
